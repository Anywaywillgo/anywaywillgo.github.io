不义联盟第五年039

太急了。本来应该更戏剧化更大场面的。

至少多抖点包袱嘛……

第39章

最后一搏

正义大厅。

“克拉克，我带来了想和你谈话的某个人……”

你想和我谈？

一个普通罪犯。我为什么不直接把你锁起来？

我是来做交易的。

我能给你蝙蝠侠的秘密藏身处。

我指望我相信你已经准备好出卖蝙蝠侠了？

你以为我是什么样的蠢货？！

// 多谢盖章，另外你就是蠢货。

最糟糕的那种。你们都是。布鲁斯，克拉克，戴安娜……

你们都是在所有人都死光之前不肯停手的蠢货。

你想要怎样？

以不再有人受伤的方式终结这场战争。

我会告诉你蝙蝠侠藏身处的位置，让你一劳永逸地解决反抗军……

……但只有你承诺不伤害布鲁斯才行。

你这么做是为了保护布鲁斯。

因为你爱他？

只管告诉我……交易成立吗？

太平洋铅矿，阿拉斯加西北部。

“跨维度传输装置已经可以运转了，莱克斯。我能够看到平行地球……”

这是个开始。但会不会奏效还要再看。

会奏效的。必须得奏效。

我们需要神奇女侠、闪电侠、绿灯侠、海王和钢骨。

不知道使用这机器我们能有几次机会，所以我们需要同时把他们全部都拉过来。

对我们来说幸运的是……他们都在一个地方。

实际上，正在进行的战斗可动静不小。

我们需要确保我们能够分离出英雄们。我们最不想要做的就是带过来更多的恶役。

我会尽力而为……但没什么能保证的。

我们开始吗？

在他们解决掉那种威胁之前就把他们拉过来可能会对他们的世界造成巨大损害。

我们需要等等。

布鲁斯！

超人来了！

有人出卖了你。

现在不重要了。

用爆鸣隧道吧莱克斯传送出这里。我去对付克拉克。

不。我们需要表明立场–

这就是表明立场。凯特和小芭能够掌控传送器。如果我们失败了，你会是这个世界阻止克拉克的最后希望。

快走。

明白了。

蝙蝠侠！

祝好运。

全然由铅包裹。够聪明的藏身处啊……

……但这也阻止不了我找到你！

出来面对我！

他走了。

我们需要让那个平行地球重新上线–

呃啊！

小芭！

布鲁斯！

出来让我们了结此事–

你想要再次一决雌雄？

呀啊啊啊！

我们必须……修好它……

我同意……但是怎么做？控制台完全被烤焦了。

我或许还是可以从这里手动操作的……

这又能运作了……

他有一个核弹头！

这就是终点，布鲁斯……

再没计划可行。

再没诡计可使。

// 光撩不给肉！无良官方饿死粉丝！

一无所有！

// 呼应第一年？脊椎上的伤不可能完全好的吧。

滚。没人喜欢电灯泡。

我们必须把小丑和他们一起传送过来……赶在他引爆核弹之前！

这就做！

// 别动我小芭。大概会再残一次。总之便当不收。

没有奏效！

你孑然一身，布鲁斯。遍体鳞伤，一败涂地。

// 谁害的？真好意思……

布鲁斯，我真的希望你能听见我这些话……

传送器被毁了。

// 抱着病态的趣味，容我再欣赏一会儿……

在这一切之后，你不可能还相信着值得这么做吧。

我们在试图修复……

面对现实吧，布鲁斯……结束了。

*……但是我们需要更多时间。*

// 我需要更多肉！
